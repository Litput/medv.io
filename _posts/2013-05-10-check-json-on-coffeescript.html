--- layout: post title: Проверка структуры JSON на CoffeeScript date: 2013-05-10 01:04 comments: true categories: [] --- Возникла задача проверки приходящего JSON от клиента на node.js. Как вы делаете эти проверки? Я реализовал простую функцию: {% highlight coffee-script %} json = key: 'value' sub: key: 'value' extra: 'value' check = expect json, {key: yes, sub: {key: yes}} console.log check # true {% endhighlight %}
